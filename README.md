# WildSprout: Skylands
A mod that adds floating islands meant for use in modpacks with create aeronautics to have some extra spice in normal worlds

## FEATURES
- sky islands
they generate as (decently rare) structures in the sky
<img width="1920" height="1080" alt="2026-06-29_14 06 25" src="https://github.com/user-attachments/assets/5d70d037-a454-474f-8ec5-e740fd94b6ae" />



## INCLUDED BIOMES
- all vanilla biomes

## CONFIGURATION
check the config folder
- max height the island can generate at
- min height the island can generate at
